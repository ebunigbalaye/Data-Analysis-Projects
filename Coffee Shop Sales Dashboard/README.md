# Coffee Shop Sales Dashboard (Excel)

This project analyzes transaction-level data from a fictional multi-location coffee shop and presents the results in an interactive Excel dashboard.

## Objectives

- Understand overall sales trends over time
- Compare performance across store locations
- Identify peak trading hours
- Analyze product category contributions to revenue

## Data

- Source: Public sample coffee shop sales dataset
- Grain: One row per transaction line item
- Key fields: Date, time, store location, product, product category, unit price, quantity

## Tools

- Microsoft Excel
  - PivotTables
  - Charts
  - Slicers
  - Dashboard layout and formatting

## Deliverables

  - KPI cards (Total Sales, Transactions, Average Order Amount)
  - Sales over time chart
  - Sales by store chart
  - Sales by hour of day chart
  - Sales by product category chart
  - Slicers for date, store, and product category

## Skills Demonstrated

- Data cleaning and validation in Excel
- Exploratory data analysis with PivotTables
- Designing interactive dashboards for business stakeholders
- Presenting insights visually for decision-making
